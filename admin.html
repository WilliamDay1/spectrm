GitHub setup
Connect to GitHub to load and push data.
One-time setup. Create a Personal Access Token at github.com → profile photo → Settings → Developer settings → Personal access tokens → Tokens (classic). Tick the repo checkbox. Copy the token and paste it below. It's saved only in this browser.
